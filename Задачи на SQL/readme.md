# Исследование базы данных сервиса Stackoverflow

<b>Постановка задачи</b>

В этом исследовании работа идёт с базой данных StackOverflow — сервиса вопросов и ответов о программировании. StackOverflow похож на социальную сеть — пользователи сервиса задают вопросы, отвечают на посты, оставляют комментарии и ставят оценки другим ответам. Необходимо исследовать предоставленные данные, сформировать требуемые выгрузки данных с помощью SQL, посчитать различные продуктовые метрики и визуализировать их.
